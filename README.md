# 🔥 IPGrabber – Ultimate IP Logger & Intelligence Tool

**IPGrabber** is a powerful Python-based tool for grabbing IP addresses, scanning networks, and collecting intelligence on targets. Perfect for penetration testing, OSINT, and social engineering campaigns.

## Features
- ✅ **IP Grabbing** – Get target IPs with a single link
- ✅ **Network Scanning** – Discover all devices on the local network
- ✅ **File Exfiltration** – Upload target files to your Discord
- ✅ **Stealth Mode** – No logs, no traces, self-destructs
- ✅ **Fake Results** – Distracts targets with realistic fake output
